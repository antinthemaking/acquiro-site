01 · Colour

Black canvas. Orange accent.

Black-and-orange brand. Orange is accent only — never decorative, never dominant. Every other tone is an extension of the dark canvas. Cap orange usage at roughly 7% of any composition.

Rules
Orange max ~7% of any composition. Reserve for: primary CTA fill on hover, hero accent word, eyebrow rules, stat callouts when emphasised, single-letter index marks. Never as section background. Never as gradient surface. Never as decorative shape. Stat numbers themselves stay cream — never orange.
02 · Typography

Inter throughout. Hierarchy by weight × scale.

Inter is the type system. No serifs, no alternates. Hierarchy comes from weight and scale, not family swaps. JetBrains Mono is reserved for eyebrows, tags, and metadata — never body copy.

05 · Navigation

Sticky. Frosted on scroll.

Logo + four links + primary CTA. Always visible, sticky on scroll. Background goes from transparent (over hero) to frosted dark (after first scroll). Primary CTA stays visible at every breakpoint.

Rules
Header is always sticky. Default state is fully transparent over the hero — the liquid background bleeds through the nav. After ~80px of scroll, the bar gains a frosted background and a hairline bottom border. Primary CTA never disappears, including on mobile (it shrinks beside a hamburger). Active link gets the orange underline rule, not the orange text colour — keeps the tracker visible without screaming.
06 · Footer

Final CTA. Then meta.

Persistent CTA strip closes the page. Below it: wordmark, contact, social, legal. Single dense row of meta — no decorative columns. The footer is the last conversion attempt before the user leaves.

07 · Motion

Motion is communication, not decoration.

Hover states reveal interactivity. Entrance animations cue the user that a section has loaded. Both stay slow and confident — never jittery. Long durations (~360ms standard, 600ms emphasised), GPU-friendly transforms only.

07.1 · BUTTON HOVER
Lift & invert

Primary button lifts 1px, fills white, text inverts to black. 200ms ease-out. Hover the button below.

07.2 · CARD LIFT
Translate-Y on hover

Cards lift 4px and the border picks up the orange. Cue: this is a clickable surface. Hover the card below.

Opacity → orange

Nav links sit at 80% opacity. On hover they go full opacity and the colour shifts to orange. 120ms — fast, responsive.

07.6 · HORIZON
The signature background

Pure dark canvas with a smooth warm bloom rising from below — bloom origin sits off-canvas, so only the soft upper edge diffuses into the dark. No hard edges, no defined band. Subtle 36s opacity pulse. Edge vignette pulls corners to near-black. Lives on the hero of every page.

LIVE PREVIEW
Purposeful
Motion communicates state or hierarchy. Never decorative.
Slow
Long durations (200–600ms). Confident, not jittery.
One thing
One animated element per viewport. The liquid bg is the exception.
Reduced
Honour prefers-reduced-motion. Static fallback for blobs & reveals.
Rules
Animate transform and opacity only — never width, height, top, or left. Use will-change sparingly, only on permanently-animated elements like the hero blobs. Stagger entrance animations by 80–160ms when blocks share a parent. Always include a prefers-reduced-motion fallback that pins the element to its end state.
